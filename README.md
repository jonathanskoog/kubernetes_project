# Kubernetes project
A BTH course development project. The project is based on the cloud and Kubernetes architecture.
Some of the technologies used in this project are:
- Docker
- Kubernetes
- Javascript
- React
- Postgres
- Typescript
- Prisma

# Installation
- Clone the repository
- cd into the kubernetes directory, cd kubernetes_project/kubernetes
- Run the run.sh file to start the Kubernetes cluster
- The application should be available on http://localhost:3000/
- When finished, run the cleanup.sh file to clean all running pods
